# Little-Lemon
This is a food delivery application that solves the online restaurant booking use case for Little Lemon Restaurant. This is a part of the Meta Certified Professional Front-end Developer Certification Program capstone project.

<h2>Installation:</h2> <br />
<p>Clone this repository using Git and then run <em>npm install</em> inside the folder. To run the project, use <em>npm start</em></p>

<h2>Tools and technologies used:</h2> <br />
<p>I have used React.js with Tailwind.css for styling the website. React-router-dom has been used for routing.</p>
